CSS 3

Es un lenguaje de hoja de estilo que ayude a controlar el aspecto o presentacion de los documentos electronicos creados con HTML al utilizar CSS se separan los contenidos de las presentacion del documento CSS se significa carcading style sheets o hoja de estilo en cascada trabaja junto con HTML para proveer estilos visuales a los elemntos del documento; tamaño, color, fondos, bordes, fuentes etc.

CSS tiene un set predeterminado de propiedades que permiten sobre escribir los estilos proevisto por navegadores y obtener la organizacion deseada. Estas propiedades no son especificas, tiene que ser combinadas para formar reglas que despues seran usadas para agrupar cajas y obtener la correcta disposicion en pantalla.

Al crear una pagina web se utiliza en el primer lugar el lenguaje HTML para marcar los contenidos es decir para designar la funcion de cada elemento dentro de la pagina Parrafos, Titulos, Tablas listas de elemento.
